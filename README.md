### idea配置git和GitHub  
1. file ==> new ==> project from version controller ==>Git  
选择repository（仓库url）和 本地项目路径  
  配置仓库的jdk maven  
  编辑仓库中的code  

2. 仓库右击 Git ==>  Add  
仓库右击 Git ==>  commit directory  
    弹框选中提交内容（不选择idea的配置文件 .idea)    填写日志信息  
    提交Commint and Push  本地仓库 GitHub仓库  
        Creat Path    
        commit      本地仓库  

3. 本地仓库 合并 GitHub仓库  
仓库右击Git ==> Repository ==> push
